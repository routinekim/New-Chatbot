# 2025학년도 하계 계절학기 운영계획(안)

## 1. 추진 배경 및 목적

가. 졸업학점 조기 이수 및 재수강 수요를 충족하기 위해 하계 계절학기를 운영한다.

나. 학사운영규정 제4조제2항에 근거하여 정규학기 외 단기 집중 수업을 제공한다.

## 2. 운영 개요

| 항목 | 내용 |
|------|------|
| 수업기간 | 2025. 6. 23.(월) ~ 7. 11.(금), 3주 |
| 1일 수업시간 | 1~4교시 (09:00~13:00) |
| 개설 예상 과목 | 교양 20과목, 전공 25과목 (총 45과목) |
| 수강료 | 학점당 80,000원(실습 95,000원) |
| 폐강 기준 | 수강인원 10명 미만 |

## 3. 세부 추진일정

1. 개설 교과목 취합: ~ 6. 2.
2. 수강신청: 6. 13. ~ 6. 16.
3. 수강료 납부: 6. 18. ~ 6. 20.
4. 폐강 확정 및 정정: 6. 20.
5. 수업 운영: 6. 23. ~ 7. 11.
6. 성적 입력: 7. 14. ~ 7. 18.

## 4. 예산 계획

가. 수입(추정): 수강료 학점당 80,000원 × 예상 3,000학점 = 240,000,000원

나. 지출(추정)

| 항목 | 금액 |
|------|------|
| 강사료(시간강사 시간당 60,000원) | 135,000,000원 |
| 운영 인건비(조교 등) | 12,000,000원 |
| 시설·냉방 등 운영비 | 18,000,000원 |
| 예비비 | 5,000,000원 |
| **합계** | **170,000,000원** |

다. 수지차(추정): 약 70,000,000원 (대학회계 전입)

## 5. 기대효과

가. 졸업 유예 감소 및 학생 학습권 보장

나. 재수강 수요 분산으로 정규학기 강의 부담 완화

## 6. 행정사항

가. 각 학과 협조 요청 공문 발송: [계절학기 개설 기안문](../04_기안문_결재공문/기안문_계절학기개설계획.md) 참조
