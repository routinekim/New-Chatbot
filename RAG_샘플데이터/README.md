## 🔎 테스트 질문 예시

### 단일 문서로 답할 수 있는 질문
- "한 학기에 최대 몇 학점까지 신청할 수 있나요?" → 학사운영규정
- "재수강하면 최고 성적이 어디까지 올라가나요?" → 학사운영규정
- "성적 이의신청은 며칠 안에 해야 하나요?" → 성적처리 지침
- "수강신청 과목 폐강 기준 인원은?" → 수강신청 매뉴얼 / FAQ

### 여러 문서를 함께 검색해야 하는 교차 질문 (RAG 성능 테스트용)
- "출장 갈 때 여비를 얼마나 받을 수 있고, 신청은 어떻게 하나요?" → 출장여비 규정 + 국내출장 기안문
- "계절학기는 어떻게 개설되고 실제 운영 결과는 어땠나요?" → 기안문 + 계획서 + 결과보고서
- "성적 정정은 어떤 사유로만 가능하고, 관련 민원은 어떻게 처리됐나요?" → 성적처리 지침 + FAQ + 7월 회의록
- "학사경고를 몇 번 받으면 제적되나요?" → 학사운영규정 + 성적처리 지침

### 표·숫자 추출 질문 (정확도 테스트용)
- "일반 교직원의 1일 일비는 얼마인가요?" → 20,000원
- "졸업이수학점은?" → 130학점
- "하계 계절학기 실제 이수학점은?" → 3,156학점

## 💡 활용 팁
- 폴더/파일명과 각 문서 상단의 YAML 메타데이터(문서종류, 소관부서, 시행일 등)를 메타필터로 활용하면 검색 정확도를 높일 수 있습니다.
- 마크다운 헤더(`##`) 단위로 청킹(chunking)하면 조항·항목별 검색이 깔끔하게 됩니다.
