# 수강신청 운영 업무 매뉴얼

## 1. 개요

이 매뉴얼은 학사팀 담당자가 매 학기 수강신청을 차질 없이 운영하기 위한 단계별 절차를 안내한다. 수강신청은 정정 포함 약 4주에 걸쳐 진행된다.

## 2. 전체 일정 (제1학기 기준)

| 단계 | 기간 | 담당 |
|------|------|------|
| 1. 개설과목 확정 | 학기 시작 8주 전 | 학과/학사팀 |
| 2. 수강편람 공시 | 학기 시작 6주 전 | 학사팀 |
| 3. 예비수강신청(장바구니) | 학기 시작 4주 전, 3일간 | 학생 |
| 4. 본 수강신청 | 학기 시작 3주 전, 학년별 3일 | 학생 |
| 5. 수강정정 | 개강 후 1주간 | 학생 |
| 6. 수강철회 | 개강 후 4주차 | 학생 |

## 3. 단계별 처리 절차

### 3.1 개설과목 확정
1. 각 학과로부터 개설 희망 과목 취합 (전공/교양 구분).
2. 폐강 기준 확인: 수강신청 인원이 **15명 미만**인 과목은 폐강을 원칙으로 한다. 단, 졸업필수·교직 과목은 예외.
3. 강의실·시간표 중복 점검 후 학사정보시스템에 등록.

### 3.2 수강편람 공시
1. 확정된 시간표를 PDF 및 시스템에 게시.
2. 학생 공지: 홈페이지 공지, 문자 발송.

### 3.3 본 수강신청 운영
1. 학년별 신청일을 분리하여 서버 부하를 분산한다(4학년 → 3학년 → 2학년 → 1학년 순).
2. 신청 학점 한도: 학사운영규정 제6조에 따라 최소 12 ~ 최대 19학점(직전 학기 평점 4.0 이상은 21학점).
3. 신청 시작 직후 30분간 학사팀 전 직원 상황실 대기 (장애 대응).

### 3.4 수강정정 및 철회
1. 정정: 과목 추가/삭제 모두 가능. 정정 후에도 12학점 이상 유지 필수.
2. 철회: 철회한 과목은 성적표에 'W'로 표기되며 학점에 포함되지 않는다. 철회 후 신청학점이 12학점 미만이 되면 철회 불가.

## 4. 자주 발생하는 민원 처리

가. **"수강신청이 마감되었는데 꼭 들어야 하는 전공이 있어요."** → 담당 교원 동의서 + 학과장 승인 시 학사팀이 수동 추가(여석 범위 내).

나. **"초과학점을 신청하고 싶어요."** → 직전 학기 성적 확인 후 기준 충족 시 시스템에서 자동 허용. 미충족 시 불가.

다. **"폐강된 과목 대신 다른 과목을 신청하고 싶어요."** → 폐강 공지 후 별도 정정기간(2일) 부여.

## 5. 체크리스트 (담당자용)

- [ ] 개설과목 시간표 중복 0건 확인
- [ ] 폐강 대상 과목 학과 통보 완료
- [ ] 서버 부하테스트 완료
- [ ] 정정기간 공지 발송
- [ ] 신청 결과 백업 저장
